Simple Calculator (Java GUI)
Project description:
It is just a calculator application created with Java Swing
It ennables the users to carry out simple arithmetic operations like:
Addition
Subtraction
Multiplication
Division
How to run the program:
Make sure that you have Java JDK installed in your system
Open the project in VS code
Navigate to the 'src' folder
Run the "CalculatorGUI.java" file.
Calculator GUI window will be opened.
